# Spotify Volume Control
Modified version of Marcin's script to control Spotify volume

Requires pactl (pulseaudio-utils on Fedora)

Raise volume 10%
```
spotifyvctl.py raise 10
```
Lower volume 5%
```
spotifyvctl.py lower 5
```
